# Groupe_DigitalBank_Partie2
Plateforme de gestion &amp; monitoring no-code

**Contexte :** Restauration, Sécurisation et Modernisation d'une Plateforme Bancaire en Ligne 

## RÉSUMÉ EXÉCUTIF

Objectif: Concevoir et déployer une plateforme no-code/low-code permettant à DigitalBank France de :
- ✅ Visualiser les données clients et transactions en temps réel
- ✅ Détecter automatiquement les activités frauduleuses
- ✅ Gérer les accès utilisateurs avec un système RBAC sécurisé
- ✅ Monitorer la sécurité et les performances du système
- ✅ Générer des rapports automatisés

**Utilisateurs cibles :** 4 personas distincts avec des besoins spécifiques

## CONTEXTE GÉNÉRAL :

### Situation Actuelle
- **Incident** : DigitalBank a subi une attaque ransomware le 15 décembre 2025
- **Conséquences** : Service interrompu 48h, perte de 7 jours de données transactionnelles
- **Données restaurées** : PostgreSQL (backup 8 déc), logs partiels, données clients
- **Infrastructure** : AWS Azure, API REST Node.js/Express, Applications React/React Native

### Besoins Stratégiques
1. **Restaurer la confiance** : Transparence et monitoring visible
2. **Prévenir les fraudes** : Détection en temps réel avec alertes
3. **Agir rapidement** : Outils no-code pour itérations rapides
4. **Assurer la conformité** : RGPD, PCI-DSS, ACPR
5. **Scaler facilement** : 850 000 clients actifs, 12M transactions/mois

### Opportunités
- Moderniser l'infrastructure avec des outils no-code
- Centraliser le monitoring et les alertes
- Améliorer la visibilité sur les opérations
- Automatiser les workflows répétitifs
